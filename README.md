# ロボアニメクイズ大会システム

## 概要
ロボスタディオンの宴会・イベント用クイズシステム

## 機能
- ⚡ 早押しクイズ（キーボード対応）
- 🎨 シルエットクイズ（段階的ヒント表示）
- 🎵 イントロクイズ（音声再生）

## 使い方

### 起動方法
```bash
cd /home/claude/robot_anime_quiz
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` にアクセス

### 操作方法

#### 司会者操作
- **Enter**: 正解表示
- **→（右矢印）**: 次の問題
- **スペース**: ヒント表示（シルエット）/ イントロ再生

#### 参加者操作
- **1〜6キー**: 早押しボタン（プレイヤー1〜6）

#### 得点管理（テスト用）
- **Shift + 1〜6**: 該当プレイヤーに10pt加算

## ファイル構成
```
robot_anime_quiz/
├── index.html          # メインファイル
├── data/
│   └── questions.json  # 問題データベース
├── images/             # シルエット画像（今後追加）
└── audio/              # イントロ音声（今後追加）
```

## 問題データの追加方法

`data/questions.json` を編集して問題を追加できます。

### 早押しクイズ
```json
{
  "id": 11,
  "question": "問題文",
  "answer": "正解",
  "difficulty": 1  // 1=初級, 2=中級, 3=上級
}
```

### シルエットクイズ
```json
{
  "id": 6,
  "robot_name": "ロボット名",
  "hint1": "ヒント1",
  "hint2": "ヒント2",
  "hint3": "ヒント3",
  "difficulty": 2
}
```

### イントロクイズ
```json
{
  "id": 6,
  "anime_title": "アニメタイトル",
  "song_title": "曲名",
  "artist": "歌手名",
  "hint": "ヒント",
  "difficulty": 2
}
```

## カスタマイズ
- プレイヤー名は `index.html` の `initPlayers()` 関数で変更可能
- デザインは `<style>` タグ内で調整可能
- 問題数は `questions.json` で自由に追加可能

## トラブルシューティング

### 問題が表示されない
- ブラウザの開発者ツール（F12）でエラーを確認
- `questions.json` の形式が正しいか確認

### キーが反応しない
- ページがアクティブか確認（クリックしてフォーカス）
- キーボードが正しく接続されているか確認
